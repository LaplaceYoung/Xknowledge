How to Sync X Bookmarks to Notion

Build a searchable Notion database directly from your X bookmarks. This guide covers schema setup, token configuration, and AI-enriched sync so each tweet becomes reusable knowledge.

Required Notion database schema

Create a Notion database with the following fields before pushing bookmarks.

Name (Title) URL (URL) Date (Date) Category (Select) Tags (Multi-select)

Sync setup in X-knowledge

  1. Create a Notion internal integration and copy the token.
  2. Share your target database with the integration.
  3. Paste token + database ID in extension settings.
  4. Open any captured bookmark and click "Push to Notion".

How teams use this

  • Product teams: capture competitor announcements and auto-tag by product area.
  • Growth teams: collect campaign references with category + tactic labels.
  • Research teams: centralize signal tweets with searchable summaries.

FAQ

  • Can this replace Notion Web Clipper? Yes for X bookmarks, with richer AI metadata and queue control.
  • Do I need cloud sync for secrets? No, local-first storage is the default mode.
  • Can I export if Notion is unavailable? Yes, markdown/JSON/ZIP remain available offline.
