How to Export X Bookmarks to Obsidian

This workflow turns your X bookmarks into durable markdown notes in Obsidian. You get AI summaries, reusable tags, and a repeatable capture pipeline for research, creator intel, and product monitoring.

Step-by-step setup

  1. Install the X-knowledge extension from GitHub and load it in Chrome developer mode.
  2. Open your X bookmarks page and let the extension capture tweet content automatically.
  3. Run AI analysis for summary and tags on single items or with batch queue controls.
  4. Use "Push to Obsidian" or export selected markdown files for vault import.
  5. Organize notes in Obsidian by date, author, and topic tags.

Best practices for clean notes

  • Keep tag schema broad and reusable (`#AI`, `#Growth`, `#Design`) instead of one-off tags.
  • Use daily folder naming (`YYYY-MM-DD`) to keep ingest consistent.
  • Batch export weekly to avoid missing content from heavy bookmark days.

FAQ

Do I need Obsidian plugins? No. The URI push works directly, and markdown export is plugin-free.
Can I preserve media references? Yes. Use ZIP export when you need offline media + markdown together.
Is data cloud synced by default? No. Sensitive keys are local-first unless you opt into cross-device sync.
